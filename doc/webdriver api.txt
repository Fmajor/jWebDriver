官方文档：http://code.google.com/p/selenium/wiki/JsonWireProtocol

Capabilities JSON Object：
	browserName						浏览器名(chrome|firefox|htmlunit|internet explorer|iphone)
	version							版本号
	platform						操作系统(WINDOWS|XP|VISTA|MAC|LINUX|UNIX)
	javascriptEnabled				JS是否被禁用
	takesScreenshot					是否允许截图
	handlesAlerts					是否能与弹出窗口交互
	databaseEnabled					是否支持数据库
	locationContextEnabled			是否能访问location上下文
	applicationCacheEnabled			是否支持应用缓存
	browserConnectionEnabled		是否能查询网络连接状态
	cssSelectorsEnabled				是否支持CSS选择器
	webStorageEnabled				是否支持storage
	rotatable						是否支持旋转
	acceptSslCerts					是否支持SSL证书
	nativeEvents					是否支持事件模拟
	proxy							代理服务器设置


Proxy JSON Object：
	proxyType						代理类型：direct(直接连接),manual(手动设置),pac(自动代理),system(使用系统连接)
	proxyAutoconfigUrl				PAC地址
	ftpProxy						FTP代理地址
	httpProxy						HTTP代理地址
	sslProxy						SSL代理地址

Cookie JSON Object：
	name	 string	 cookie名字
	value	 string	 cookie值
	path	 string	 路径(可选)
	domain	 string	 域名(可选)
	secure	 boolean	是否加密(可选)
	expiry	 number	 过期时间(可选)

Responses：
	sessionId						会话ID
	status							状态码
	value							响应值

Response Status Codes：
	0	Success						成功
	7	NoSuchElement				无对应对象
	8	NoSuchFrame					无frame
	9	UnknownCommand				未知命令
	10	StaleElementReference		旧对象引用
	11	ElementNotVisible			对象不可见
	12	InvalidElementState			对象不可用
	13	UnknownError				未知错误
	15	ElementIsNotSelectable		对象不可选择
	17	JavaScriptError				JS错误
	19	XPathLookupError			XPath查询错误
	21	Timeout						超时
	23	NoSuchWindow				无指定窗口
	24	InvalidCookieDomain			无效Cookie域名
	25	UnableToSetCookie			无法设置Cookie
	26	UnexpectedAlertOpen			意外Alert弹出
	27	NoAlertOpenError			无Alert弹出错误
	28	ScriptTimeout				代码执行超时
	29	InvalidElementCoordinates	无效对象坐标
	30	IMENotAvailable				不支持输入法
	31	IMEEngineActivationFailed	输入法设置失败
	32	InvalidSelector				无效选择器(Xpath,CSS)
	33	SessionNotCreatedException	会话没有建立
	34	MoveTargetOutOfBounds		对象移动溢出
